# Homebrew Tap for Tokemon

Monitor your Claude Code usage from the macOS menu bar.

## Installation

```bash
brew tap AverageHelper/tokemon
brew install --cask tokemon
```

## Upgrading

```bash
brew upgrade --cask tokemon
```

## Uninstalling

```bash
brew uninstall --cask tokemon
brew untap AverageHelper/tokemon
```

## About

Tokemon shows your Claude Code session usage, alerts before hitting limits, and helps you pace your work. It runs as a lightweight menu bar app on macOS 14+.

- Real-time token usage in your menu bar
- Configurable alerts before hitting limits
- Multiple display styles (icon, progress bar, battery)
- Multi-profile support for team accounts

[Learn more at tokemon.app](https://tokemon.app)

## Requirements

- macOS 14.0 (Sonoma) or later

## Troubleshooting

If macOS shows a security warning, ensure you have the latest version:

```bash
brew upgrade --cask tokemon
```

The app is signed with a Developer ID certificate and notarized by Apple.
