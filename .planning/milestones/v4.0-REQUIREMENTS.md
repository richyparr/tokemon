# Requirements Archive: v4.0 Raycast Integration

**Archived:** 2026-02-24
**Status:** All 18 requirements complete

## Requirements

### Extension Foundation

- [x] **EXT-01**: Extension scaffolded with TypeScript, React, @raycast/api
- [x] **EXT-02**: Custom Tokemon icon displayed in Raycast
- [x] **EXT-03**: MIT license and README with setup instructions
- [x] **EXT-04**: User can enter OAuth token via setup wizard

### Dashboard Command

- [x] **DASH-01**: User sees session usage percentage
- [x] **DASH-02**: User sees weekly usage percentage
- [x] **DASH-03**: User sees reset timer countdown
- [x] **DASH-04**: User sees pace indicator (on track / ahead / behind)
- [x] **DASH-05**: User can manually refresh usage data

### Menu Bar Command

- [x] **MENU-01**: Usage percentage displayed in Raycast menu bar
- [x] **MENU-02**: Menu bar updates automatically (background refresh)
- [x] **MENU-03**: Menu bar color indicates usage level (green/orange/red)

### Multi-Profile

- [x] **PROF-01**: User can add multiple profiles with tokens
- [x] **PROF-02**: User can switch between profiles
- [x] **PROF-03**: User can delete profiles

### Alerts

- [x] **ALRT-01**: User can configure usage threshold for alerts
- [x] **ALRT-02**: User receives Raycast notification at threshold
- [x] **ALRT-03**: User can test alert from settings
